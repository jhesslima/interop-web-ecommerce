## 🚀 Como Visualizar o Projeto

### 🖥️ Rodando Localmente

Para rodar o projeto na sua própria máquina, siga os passos:

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/jhesslima/interop-web-ecommerce.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd XMLHttpRequest/
    ```
3.  **Abra o arquivo principal:**
    * Simplesmente abra o arquivo principal no seu navegador de preferência (Google Chrome, Firefox, etc.).
